How it Works

Generate: Creates a random password using the inputs entered, then any unaccounted for characters in the length will be generated as random lowercase characters. In order to generate a password; the length must be greater than zero and the number of characters cannot exceed the length. All empty fields other than the length will default to zero if left empty.

Generic: Creates a random password of 14 - 16 characters and a random number of characters from each field.
